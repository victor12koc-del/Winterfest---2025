# Winterfest 2025 Website

This is the official website for Winterfest 2025, hosted by XIME Kochi.

## Project Structure
- `index.html`: The main website file.
- `style.css`: Contains all the styling, animations, and responsive design.
- `script.js`: Handles the snow effect, countdown timer, and interactions.
- `snowglobe_main.jpg`: The main hero image.

## How to Host on GitHub Pages

1.  **Create a Repository**: Go to GitHub and create a new repository (e.g., `winterfest-2025`).
2.  **Upload Files**: Upload all the files in this folder (`index.html`, `style.css`, `script.js`, `snowglobe_main.jpg`) to the repository.
3.  **Settings**:
    -   Go to the repository **Settings**.
    -   Click on **Pages** in the left sidebar.
    -   Under **Build and deployment**, select **Source** as `Deploy from a branch`.
    -   Select the branch (usually `main` or `master`) and folder `/(root)`.
    -   Click **Save**.
4.  **Live Link**: Wait a minute, and GitHub will provide you with a link (e.g., `https://yourusername.github.io/winterfest-2025/`).

## Features
-   **Interactive Snowfall**: Snow falls faster as you scroll.
-   **Countdown Timer**: Live countdown to Nov 28, 2025.
-   **Responsive Design**: Works on mobile and desktop.
-   **Event Details**: Comprehensive list of management, sports, and cultural events.
